# vue-js-client-crud
Simple CRUD client built with axios and vuetify. Communicates with simple-nodejs-express-mongodb.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
